## 使用说明
1. 确保 要筛选的表格是 .csv 格式
2. 确保 要筛选的表格没有被其他软件打开
3. 确保 headline处于表格中的第一列
4. 50万条数据占用内存近1G，请确保电脑剩余内存足够

5. 软件闪退的解决方法
     - 打开cmd，将软件拖进去打开，查看错误代码
     - 闪退多是因为文档编码问题，将文档以UTF-8编码保存再重试

6. 第一次打开软件会在同级目录下生成keywords.ini文件
     - 内置了24个疑问词，可以自己添加，一行一个
